# Учебный проект #

## Сайт для поиска работы ~~вкатунам~~ опытным IT-шникам!

___



### Описание проекта:
- На сайте любой пользователь может просматривать вакансии.  
Использую форму поиска или группировку для него подготовленную.
- У пользователя есть возможность регистрации и авторизации.
- Авторизированные пользователи имеют возможность создать своё  
резюме и редактировать его.
- Авторизированные пользователи имеют возможность зарегистрировать свою компанию.
- Если пользователь зарегистрировал свою компанию, может так-же создавать  
и редактировать для неё вакансии.
- Авторизированные пользователи могут откликнуться на нужную вакансию заполнив форму под ней.  
у пользователя который эту компанию зарегистрировал, есть возможность просмотреть кандидатов  
откликнувшихся на вакансию.
___

### Установка:

#### Проект использует версию python 3.12.7

1) Быстрый способ поставить проект к себе через вашу IDE.  
Скопируйте и вставьте у себя в разделе Clone Repository:
```
https://github.com/Andrew-Tyurin/work_for_vkatuns.git
```
Таким способом ваша IDE должна сома подтянуть все пакеты  
и поставить виртуальное окружение.


2) Через терминал:

*Следующие команды вводим из корня проекта:*

- В корне проекта создаём виртуальное окружение
```bash
python3.12 -m venv .venv
```

- Затем, активируем его
```bash
source .venv/bin/activate
```

- Ставим зависимые пакеты
```bash
pip install -r requirements.txt
```

- После чего запускаем проект(доступен по: http://localhost:8000/ или http://127.0.0.1:8000/)
```bash
python manage.py runserver
```
